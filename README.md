Laravel with JWT Auth + ReactJS 

Tools :
1. Laravel
2. JWT Auth
3. ReactJS
4. Semantic UI
--------------

1. Rename .env.example to .env
2. Update DB_HOST, DB_DATABASE, DB_USERNAME, DB_PASSWORD.
3. In command line : 
   - composer.phar update
   - Php artisan key:generate
   - php artisan migrate
4. npm install
5. npm run dev
